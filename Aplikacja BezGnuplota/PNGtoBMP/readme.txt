Introduction
------------
Easy2Convert PNG to BMP is a small freeware 
utility for converting Portable Network Graphics files (.png) 
to Windows or OS/2 Bitmap files (.bmp) easily and quickly.

System requirements
-------------------
 * Windows 2000/XP/2003/Vista/7/8
 * less than 3 Mb of free space on hard disk
 
Installation
------------
Run an installer/unzip all files to any folder and run 
the executable file (usually, "png2bmp.exe").

Getting started
---------------
1. Select a png-file to convert
2. Select an output bmp-file
3. Click on the "Convert" button.
